# game_car
大学时候做的小玩具， 需要的库有:   
```easygui```   
```pygame```   
大致是一个汽车自动行驶，在路口遇到路标指示，根据不同的指示选择不同的前进方向。   
最终也有```pyinstaller```将其转换成exe格式版本
